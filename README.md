# 鱼跃呼吸机数据分析工具 (Yuwell CPAP Data Analyzer)

这是一个开源的数据分析工具，旨在帮助用户读取和可视化鱼跃 (Yuwell) 呼吸机（如 YH-360A）生成的 `.BYS` 数据文件。

本项目仅供学习和参考，旨在帮助用户更好地理解自己的设备使用数据。

## ✨ 主要功能

*   **一键解析**: 自动识别并解析二进制数据文件。
*   **数据导出**: 将难以阅读的二进制数据转换为通用的 CSV 表格文件。
*   **可视化报告**: 生成包含交互式图表的 HTML 报告，直观展示治疗趋势。
*   **关键指标分析**:
    *   📅 使用时长 (Duration)
    *   💨 治疗压力 (Pressure)
    *   💤 呼吸暂停指数估算 (AHI Estimate)
    *   🎭 漏气量估算 (Leak Rate Estimate)

## 📂 文件说明

*   `yuwell_analyzer.py`: **核心脚本**。集成了文件解析、数据分析和报告生成的所有功能。
*   `summary.csv`: 脚本运行后生成的原始数据记录表。
*   `report.html`: 脚本运行后生成的交互式分析报告（推荐直接打开此文件）。

## 🚀 快速开始

### 1. 准备工作
确保您的电脑上安装了 Python 环境。

### 2. 运行脚本
将呼吸机 SD 卡中的 `.BYS` 文件（通常命名为 `YHSD-NEW.BYS`）复制到脚本所在目录，然后在终端中运行：

```bash
python yuwell_analyzer.py
```

### 3. 查看结果
运行完成后，脚本会自动生成以下文件：

1.  **打开 `report.html`**：
    *   双击即可在浏览器中查看完整的图表分析。
    *   包含使用时长趋势、压力变化、AHI 指数和漏气量等关键指标的可视化。
    *   每个图表都配有详细的中文说明，帮助您理解数据的含义。

2.  **查看 `summary.csv`**：
    *   如果需要进行更深入的自定义分析（如使用 Excel），可以打开此文件。

## 📊 数据指标说明

本工具提取的数据基于文件结构分析，以下指标仅供参考：

| 指标名称 | 说明 | 正常参考范围 |
| :--- | :--- | :--- |
| **使用时长** | 单次治疗的持续时间 | 建议每晚 > 4 小时 |
| **治疗压力** | 设备输出的平均气道正压 | 根据医生处方设定 (通常 4-20 cmH2O) |
| **AHI (估算)** | 每小时呼吸暂停/低通气次数 | 正常 < 5 次/小时 |
| **漏气量** | 面罩或管路的漏气速率 | 建议 < 60 L/min |
